## üá¨üáß English Description

**Group Po'Vuxa** is a module for Foundry VTT that revolutionizes how you move your party. Instead of moving tokens individually, gather them into a single formation, move as a unit, and deploy them strategically when combat starts!

### Key Features
- **Gather & Disperse**: Condense your party into a single "Leader" token for exploration, and scatter them back when needed.
- **Formations**: Choose from tactical presets like **Line**, **Wedge**, **Square**, **Circle**, or create your own custom arrangements.
- **Smart Placement**: The module respects walls and collisions when deploying tokens.
- **Stealth Mode**: Roll Stealth for the entire group with a single click (takes the lowest roll).
- **Shared Vision**: The group token inherits vision and light sources from all members.

### Installation
1. Open Foundry VTT.
2. Go to **Add-on Modules** -> **Install Module**.
3. Paste the following Manifest URL:
   `https://github.com/ArthurPierce23/group-povuxa/releases/latest/download/module.json`
4. Click **Install**.

### Usage
1. **Select** the tokens you want to group.
2. Open the **Party Panel** (button on the left token controls).
3. Click **Gather Party**. A new group token will be created.
4. Move the group token freely.
5. To disband, select the group token and click **Disperse** or choose a formation.
